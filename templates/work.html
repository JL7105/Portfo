001 : Secure Password Checker

Data leaks are a huge problem in today's world. This project aims towards notfying users if their emails or usernames have been leaked. Through the CLI, the user will enter the email or username wanted to be checked. Then, the program applies K anonymity by converting the input to unicode and then encrypting it with SHA-1 hash encryption. This protects the user from data leaks via the API used in this program to find leaks on all other websites. This hash encryption is separated into a five char head and then all other chars tail. The program enter the encrypted head into the API to then get a list of all inputs matching the five char head. Returning to the CLI, the program then takes the tail and run through the entire list until there is a match. Once matched, the program then relays the info from the API letting the user know if their email or password has been leaked before with the number of times leaked.
